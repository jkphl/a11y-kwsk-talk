#### Tastaturkürzel

* `Bild Oben`, `←`, `↑`: Vorherige Folie
* `Leerzeichen`, `Zeilenumbruch`, `Bild Oben`, `→`, `↓`: Nächste Folie
* `c`: Mauszeiger ein- / ausblenden
* `i`: Interaktiven Modus aktivieren / deaktivieren (schrittweises Einblenden von Listenelementen)
* `Esc`: Folienübersicht
